NODEJS Kotori

A simple and open-source Discord Bot for your server.

Download this project as a .zip file Download this project as a tar.gz file

Features.

NODEJS Kotori can perform simple administrative commands and a few extra things. she can even gather the latest news!

Adding to your Discord server.

I have made adding the bot to your server easier using suckybots by abal.

  • for stable : http://suckybots.abal.moe/bots/189963740031090688
  • for Cutting Edge (Unstable) :http://suckybots.abal.moe/bots/189964084597227523
